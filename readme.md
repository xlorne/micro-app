# Mirco-App

This is a micro-app that uses the webpack5 & react & typescript to build a simple app.

## How to run

* start mirco-app
```shell
cd mirco-app
yarn
yarn start
```

* start mico-home
```shell
cd mirco-home
yarn
yarn start
```

## How to build & deploy
```shell
cd ./scripts
sh package.sh
# deploy 192.168.3.200
sh deploy.sh
```

## Micro-App test
```shell
cd mirco-app
yarn
yarn server
```
